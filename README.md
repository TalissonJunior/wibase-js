# WIBASE-JS

This is the wibase core frontend javascript, that allows you to manipulate your data..

## Development Hints for working on wibase-js


### Prerequisites

Node.js 
 
>First install [Node.js](https://nodejs.org/), recomended to download any stable version above 8.11.1


### Installing Dependencies


```sh
npm install 
```

### Running the project.


To serve the application and watch for changes use. 
```sh
npm run serve
```

an then:

```sh
npm run start
```

To buil the application:

```sh
npm run build
```
